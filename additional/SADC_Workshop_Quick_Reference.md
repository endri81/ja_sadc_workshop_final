## 📝 LAB SCRIPTS

```
"LAB SCRIPT 1.3"     → Design Effect Calculation (R + STATA)
"LAB SCRIPT 1.4"     → Optimal Allocation
"LAB SCRIPT 2.2"     → svydesign Configuration
"LAB SCRIPT 3.3"     → Calibration Methods
...etc
```

**REMEMBER:** Always run locally → verify output → then request slides

## 🖼️ SLIDE CREATION

```
"SLIDES 23-35 DAY 1 MODULE 3"
```
→ Delivers LaTeX batch + instructor notes + quality checklist

**Batch limit:** 15 slides maximum per request

## 🔧 CORRECTIONS

```
"FIX SLIDE 28: equation overflow"

```
